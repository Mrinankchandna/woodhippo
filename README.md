# WoodHippo Services

A service provider website built with Angular, inspired by BVG India.

## Features

- Modern, responsive design
- Angular standalone components
- Angular routing
- Service provider focused content

## Pages

- Home
- About Us
- Services
- Sectors
- Careers
- CSR
- Media
- Contact Us

## Development

### Prerequisites

- Node.js and npm

### Installation

```bash
npm install
```

### Development server

```bash
ng serve
```

Navigate to `http://localhost:4200/`.

### Build

```bash
ng build
```

The build artifacts will be stored in the `dist/` directory.

## Deployment

The application can be deployed to any static hosting service or server that supports Angular applications.
